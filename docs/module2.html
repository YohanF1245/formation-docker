Module 2 - Installation de Docker

Objectifs du module

  • Installer Docker sur différents systèmes d'exploitation
  • Configurer Docker pour un usage optimal
  • Vérifier l'installation et résoudre les problèmes courants

Prérequis système

Avant d'installer Docker, assurez-vous que votre système répond aux exigences minimales :

Windows

  • Windows 10 64-bit
  • WSL 2 activé
  • Virtualisation activée

MacOS

  • MacOS 10.15+
  • Processeur Intel/Apple
  • 4GB RAM minimum

Linux

  • Kernel 3.10+
  • systemd
  • 64-bit requis

Installation sur Windows

Pour installer Docker sur Windows :

  1. Activez WSL 2 via PowerShell en administrateur :
    2. dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
  2. Téléchargez Docker Desktop depuis le site officiel
  3. Exécutez l'installateur
  4. Redémarrez votre ordinateur

Installation sur MacOS

Pour installer Docker sur MacOS :

  1. Téléchargez Docker Desktop pour Mac
  2. Ouvrez le fichier .dmg
  3. Glissez l'application dans le dossier Applications
  4. Lancez Docker Desktop

Installation sur Linux

Pour installer Docker sur Ubuntu :

# Mise à jour des paquets

sudo apt-get update

# Installation des prérequis

sudo apt-get install ca-certificates curl gnupg lsb-release

# Ajout de la clé GPG officielle de Docker

curl -fsSL https://download.docker.com/linux/ubuntu/gpg | sudo gpg --dearmor -o /usr/share/keyrings/docker-archive-keyring.gpg

# Installation de Docker

sudo apt-get install docker-ce docker-ce-cli containerd.io

Vérification de l'installation

Pour vérifier que Docker est correctement installé :

docker --version
docker run hello-world
